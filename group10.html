This is a static timing analysis tool that gets the Verilog gate level net list of a digital circuit, the library of the cells used in the circuit, the net capacitances file, the clocks skew file and timing constraints (the clock period, input delays and output delays). The tool checks all the timing paths and reports the violating timing paths, if any.
